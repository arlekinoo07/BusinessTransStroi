# Bitrix24 -> Qdrant Sync

Скрипт забирает компании и сделки из Bitrix24 по webhook, превращает каждую сущность в текстовый документ, строит мультиязычные эмбеддинги локальной моделью и загружает точки в Qdrant.

## Подготовка

1. Установите зависимости:

```bash
npm install
```

2. Заполните переменные окружения:

```bash
cp .env.example .env
```

Нужны как минимум:

- `QDRANT_URL`
- `BITRIX24_WEBHOOK_URL` уже заполнен вашим webhook

`QDRANT_API_KEY` нужен только если ваш Qdrant защищен ключом.

## Запуск

```bash
npm run sync
```

По умолчанию создается коллекция `bitrix24_records`, если ее еще нет.
